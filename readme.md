# Generative Art Gallery Canvas

A Generative Atr Gallery Canvas project to train on Node.js, Electron.js, Javascript, TypeScript, p5.js, paper.js, THREE.js...
 
## lib used
- node.js
- electron.js
- express.js
- pug template engine
- Bulma css framework
